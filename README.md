# SpringBoot-MemberShip-Core
SpringBoot MemberShip Core Project ( OAuth2 + SSO + JPA(Hibernate) + DATARest ) Server

프로젝트 개요
---------------------
본 프로젝트는 *외부 social login* 도 지원하면서 *자체적인 회원관리* 도 가능한 구조이다.
